Makefile template

File Directory

Automatic Variables

  • $@: The file name of the target of the rule
  • $<: The name of the first prerequisite
  • $^: The names of all the prerequisites, with spaces between them
  • $(@D): The directory part of the file name of the target

Functions for file name

  • wildcard: Wildcard expansion happens automatically in rules
    • SRCS := $(wildcard *.c)
  • addprefix: The argument names is regarded as a series of names, separated by whitespace
    • DEFINE_FLAG := addprefix(-D, $(DEFINE_SYM))
  • patsubst: Finds whitespace-separated words in text that match pattern and replaces them with replacement
    • OBJS := $(patsubst %.c, $(OBJDIR)/%.o, $(SRCS))

Dependency file

  • 主要用來針對 header filer 改動時相關的 object 也會被重新編譯
  • 可透過gcc -MMD 產生 .d dependency
    • obj/liba.d content: obj/liba.o: liba.c libb.h liba.h
  • 可透過 -include 在 Makefile 在專案初始化時還沒有 dependency file 時先跳過檢查
